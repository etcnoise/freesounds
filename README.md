# freesounds

A small set of Creative Commons samples from [freesound.org](https://freesound.org).

## Licensing

These samples come from Freesound and use a mix of **CC0** and **CC‑BY** licenses.
- **CC0** → free to use, remix, and share with **no attribution required**  
- **CC‑BY** → free to use, remix, and share as long as you give credit  
  (the exact attribution text is already in the folder’s `LICENSE.md`)
